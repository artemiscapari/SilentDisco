Required software:
1. Python version 2.7
2. Numpy
3. OpenCV version 2.7
The different python files in the GitHub repository consist of:
1. Functions.py: All the functions used in the other scripts including functions for Image processing, 
videoprocessing, headphone detection using contour detection, headphone detection using SIFT.
2. Image_Processing_DContours.py: the executable python file to run the double contour algorithm
3. Save_Cropped_Images.py: the executable python file to run the script to save individual photos of
headphones from a larger image.
4. Run_Sift.py: The executable python file to run sift from with a directory of example images of 
head-phones to detect key points.
